accessibility:
  - "Ensuring WCAG 2.1 AA compliance was foundational across all five brand skins."
  - "Semantic HTML and ARIA landmarks kept product, store finder, and events templates navigable by screen reader."
  - "Color contrast validated independently for each sub-brand's palette before launch."
  - "Keyboard-accessible carousels and filters replaced mouse-only interaction patterns."
  - "Alt text and focus order standardized across every localized microsite."
designSystem:
  - "Atomic component library scaled across one parent and four sub-brands from a single source."
  - "Six-breakpoint responsive scale authored once per module, inherited by every brand."
  - "Adaptive content blocks let each sub-brand localize copy, imagery, and styling without forking the template."
  - "Mapbox Studio integration themed to match each brand's visual identity."
  - "Reusable page templates — product, store finder, events, about — cut new-microsite build time significantly."
  - "Token-based theming isolated brand-specific color, type, and logo swaps from shared layout logic."
tools:
  - "Figma"
  - "Sketch"
  - "Mapbox"
aboutProject:
  - "An exercise in automation, variation, and identity, this design eloquently solves a need for page templates, reusable components, with a wide range of data variation and customization capabilities."
  - "Reusability & repeatability. A modular framework comprised of adaptive content blocks is key to the template design. This modularity provides a fundamental structural consistency across Foot Locker Inc.'s parent brand experience, while giving each sub-brand control over specific data and styling needs."
